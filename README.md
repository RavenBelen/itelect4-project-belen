# ITELECT4 TypeScript Project

## Project Description

This project demonstrates the basic concepts of TypeScript, including:

- Interfaces
- Generic Interfaces
- Generic Functions
- Utility Types
- Enums

## Project Structure

```
src/
types/
package.json
tsconfig.json
```

## How to Run

Install dependencies:

```bash
npm install
```

Check for TypeScript errors:

```bash
npx tsc --noEmit
```

Compile the project:

```bash
npx tsc
```

Run the project:

```bash
node src/index.js
```
